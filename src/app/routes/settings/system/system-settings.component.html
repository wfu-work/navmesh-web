配置会写入服务端 settings 表 域名、监听地址和限制策略会影响设备接入、SSH 连接和 HTTP 映射。保存前请确认反向代理、防火墙和客户端配置已经同步。

接入域名

用于生成设备的 SSH 和 HTTPD 接入地址。设备类型仍可维护默认映射域名，单台设备也可以在设备配置中覆盖。

Gateway
SSH 网关域名
设备配置页会基于这个域名生成 SSH 登录示例，例如 device.sshd.navfirst.com。
 HTTPD 网关域名
用于规划 HTTP 映射的统一接入域名，例如 device.httpd.navfirst.com。
 设备注册 Token
只用于设备首次注册。设备注册成功后会使用独立 Token，建议定期轮换这个值。
SSH 示例 device.sshd.navfirst.com 指向 SSH 网关，设备配置页会生成登录命令。
HTTPD 示例 device.httpd.navfirst.com 可作为 HTTP 映射统一入口，再由网关转发到设备 Web 服务。
部署提醒 请确保 DNS、反向代理和证书覆盖这些域名；修改域名后，已复制出去的访问地址需要同步更新。

网关监听

控制服务端对外开放的 SSH、HTTP 和 QUIC 隧道入口。监听地址通常由部署环境决定。

Entrypoint
SSH 监听地址
用于 SSH Gateway 接入。填写 :3010 表示监听所有网卡的 3010 端口。
 SSH 网关
关闭后新 SSH 连接将无法进入网关，已有连接是否断开取决于服务端运行状态。
 HTTP 监听地址
用于接收 Caddy 或其他反向代理转发过来的 HTTP 映射请求。
 HTTP 映射网关
关闭后自定义域名和默认映射域名都不会继续转发到设备。
 隧道监听地址
设备通过 QUIC 隧道连接到这里。请确保 UDP 端口已放行。
 设备隧道
关闭后设备将无法建立新的隧道，SSH 和 HTTP 映射也会失去在线通道。

在线判定与运行限制

用于控制设备离线判断、会话回收和连接保护。数值为 0 表示不限制。

Runtime
设备心跳超时
超过这个时间没有心跳或隧道活动后，设备会被标记为离线。
 离线扫描间隔
后台按这个间隔检查设备在线状态。间隔越短，状态更新越及时。
 客户端下载基址
留空时按当前 API 域名生成，例如 /api/downloads/releases/latest?releaseType=navmesh。
 客户端升级
关闭后已创建的升级任务会暂停下发，客户端不会自动替换二进制。
 会话空闲超时
连接长时间无数据传输时自动回收，避免空闲会话占用资源。
 最大并发会话
限制全局同时在线的 SSH/HTTP 会话数量，0 表示不限制。
 单设备最大会话
限制单台设备同时承载的会话数量，适合保护低性能设备。
 来源 IP 每分钟新建会话
限制同一来源 IP 每分钟新建会话数量，降低异常请求冲击。
